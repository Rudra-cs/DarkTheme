# DarkTheme

## This app has the code for choosing the theme by the users:
  1. Light
  2. Dark
  3. System Default Theme
 
 ## Libraries & Modules Used:
  1. Shared Preference:-> to store the theme settings choosen by the user
  2. Material UI
  3. View Binding
